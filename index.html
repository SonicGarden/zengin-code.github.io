Zengin Code API

統一金融機関コードのJSON APIです。プログラムから金融機関コードと支店コードの情報を取得できます。

API エンドポイント

全金融機関一覧

GET /api/banks.json

全ての金融機関の情報を取得します。

curl https://zengin-code.github.io/api/banks.json

特定金融機関の支店一覧

GET /api/branches/{金融機関コード}.json

指定した金融機関の全支店情報を取得します。

curl https://zengin-code.github.io/api/branches/0001.json

レスポンス形式

banks.json の例

{
  "0001": {
    "code": "0001",
    "name": "みずほ",
    "kana": "ミズホ",
    "hira": "みずほ",
    "roma": "mizuho"
  },
  "0005": {
    "code": "0005",
    "name": "三菱ＵＦＪ",
    "kana": "ミツビシユーエフジェイ",
    "hira": "みつびしゆーえふじぇい",
    "roma": "mitsubishiufj"
  }
}

branches/{code}.json の例

{
  "001": {
    "code": "001",
    "name": "東京営業部",
    "kana": "トウキョウエイギョウブ",
    "hira": "とうきょうえいぎょうぶ",
    "roma": "tokyoeigyobu"
  },
  "002": {
    "code": "002",
    "name": "横浜支店",
    "kana": "ヨコハマ",
    "hira": "よこはま",
    "roma": "yokohama"
  }
}
注意: このAPIは静的ファイルとして提供されています。CORS制限はありませんので、ブラウザからも直接アクセス可能です。

プログラミング言語別ライブラリ

より簡単に統一金融機関コードを利用するための言語別ライブラリも提供しています：

  • Ruby gem - gem install zengin_code
  • JavaScript/Node.js - npm install zengin-code
  • Python - pip install zengin_code

データ更新

データは毎日自動的に更新されます。最新の情報については公式の金融機関コード表をご確認ください。
